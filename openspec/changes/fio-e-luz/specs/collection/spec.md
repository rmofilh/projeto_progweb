# Coleção Temática (Collection)

## ADDED Requirements

### Requirement: Thematic Agregation

- The system MUST aggregate patterns and stitch suggestions into thematic collections.
- The system MUST NOT allow users to create or modify the thematic dimensions arbitrarily.

#### Scenario: Viewing a thematic collection
Given the user accesses a collection (e.g., "Beginners")
When the collection details are loaded
Then the system MUST display the curated patterns
And the system MUST display the associated stitch suggestions
